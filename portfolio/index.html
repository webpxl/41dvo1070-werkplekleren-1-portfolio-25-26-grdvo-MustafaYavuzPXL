Profiel DVO & WPL1 reflectie

Een korte situering van waar ik nu sta als DVO-student en wat ik geleerd heb tijdens WPL1.

Situering binnen profiel DVO

Binnen Digital Design & Development wil ik uitgroeien tot iemand die zowel creatief kan ontwerpen als technisch kan bouwen. Ik vind het leuk om:

  • na te denken over hoe een gebruiker een website ervaart;
  • te experimenteren met kleuren, layout en animaties;
  • met HTML, CSS en JavaScript dingen tot leven te brengen.

Ik zie mezelf later in een rol waar ik kan meedenken over het concept, een design kan uitwerken en mee kan bouwen aan de front-end.

Eindreflectie WPL1

Wat ik geleerd heb

Tijdens WPL1 heb ik vooral gemerkt hoe belangrijk planning en communicatie zijn. In het begin onderschatte ik soms hoe lang dingen zouden duren, maar door WPL1:

  • plan ik mijn werk beter in kleinere taken;
  • durf ik vaker vragen te stellen als iets niet duidelijk is;
  • ben ik bewuster bezig met deadlines.

Mijn groei

Technisch gezien voel ik dat mijn basis in HTML, CSS en JavaScript sterker is geworden. Ik begrijp beter hoe een project wordt opgebouwd en hoe versiebeheer met Git en GitHub in een workflow past.

Ook op persoonlijk vlak ben ik gegroeid: ik neem sneller initiatief en ben minder bang om fouten te maken, omdat ik zie dat fouten deel zijn van het leerproces.

Vooruitblik

In volgende projecten wil ik:

  • meer aandacht besteden aan toegankelijkheid en usability;
  • complexere JavaScript-functies proberen bouwen;
  • mijn portfolio verder uitbreiden met projecten waar ik trots op ben.
